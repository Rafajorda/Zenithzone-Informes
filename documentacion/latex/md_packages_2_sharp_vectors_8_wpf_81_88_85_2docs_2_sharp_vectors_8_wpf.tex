This project provides a C\# library for parsing, converting and viewing the SVG files in WPF applications.

More documentation is available at the \href{https://elinamllc.github.io/SharpVectors/}{\texttt{Shape\+Vectors Github Pages}}.\hypertarget{md_packages_2_sharp_vectors_8_wpf_81_88_85_2docs_2_sharp_vectors_8_wpf_autotoc_md7}{}\doxysection{\texorpdfstring{Key Features}{Key Features}}\label{md_packages_2_sharp_vectors_8_wpf_81_88_85_2docs_2_sharp_vectors_8_wpf_autotoc_md7}

\begin{DoxyItemize}
\item Parsing of the SVG files to create the SVG DOM
\item SVG to XAML conversion
\item A small runtime library to handle font URI, embedded images and others when using the XAML files directly from disk.
\item An optimized XAML output.
\item Interaction with the conversion process (by a visitor pattern) to allow for custom hyper-\/link implementations, font substitutions etc.
\item Supports .NET Framework 4+, .NET 6.\+0, .NET 8.\+0 and .NET 9.\+0.
\end{DoxyItemize}\hypertarget{md_packages_2_sharp_vectors_8_wpf_81_88_85_2docs_2_sharp_vectors_8_wpf_autotoc_md8}{}\doxysection{\texorpdfstring{Main Components}{Main Components}}\label{md_packages_2_sharp_vectors_8_wpf_81_88_85_2docs_2_sharp_vectors_8_wpf_autotoc_md8}
The library consists of several components as listed below and there is no dependency on external components:\+


\begin{DoxyItemize}
\item {\ttfamily Sharp\+Vectors.\+Core.\+dll} (SVG interfaces and basic types)
\item {\ttfamily Sharp\+Vectors.\+Css.\+dll} (CSS interfaces and types)
\item {\ttfamily Sharp\+Vectors.\+Dom.\+dll} (SVG DOM, integrates CSS DOM)
\item {\ttfamily Sharp\+Vectors.\+Model.\+dll} (SVG model, parsing and utility classes)
\item {\ttfamily Sharp\+Vectors.\+Runtime.\+Wpf.\+dll} (An independent component with scroling/\+zooming panels for viewing rendered XAML)
\item {\ttfamily Sharp\+Vectors.\+Rendering.\+Wpf.\+dll} (WPF rendering of the SVG Model)
\item {\ttfamily Sharp\+Vectors.\+Converters.\+Wpf.\+dll} (Utitlity classes providing access to the WPF rendering features)

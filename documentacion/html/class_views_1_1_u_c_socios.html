UCSocios.

UserControl para la gestión de Socios Proporciona la interfaz de usuario para crear, editar, visualizar y eliminar socios Incluye gestión del estado activo/inactivo de los socios El DataContext se establece automáticamente desde el MainViewModel cuando se navega a esta vista.
